SparkDeck Database Test

Phase 1: Verify Neon Postgres connection

Before you click:

1 Make sure your .env.local file has your real DATABASE_URL
2 You must be running this via netlify dev (not npm start)
3 The URL should be http://localhost:8888/test-db.html

How to run this:

1 Open a terminal in your project folder
2 Run: npx netlify dev
3 Wait for it to say "Server now ready"
4 Open http://localhost:8888/test-db.html in your browser
5 Click the button above!
