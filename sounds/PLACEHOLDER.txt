This directory is for sound effects used in the StudyFlow app.

The app expects the following MP3 files in this directory:
- flip.mp3
- enter-study.mp3
- exit-study.mp3
- correct.mp3
- wrong.mp3

Please see README.md in this directory for instructions on where to download
royalty-free sound effects and how to add them to this folder.

The app will work without these files - sounds will simply be skipped if
the files are not present.
